# assign-to-admin

> A GitHub App built with [Probot](https://github.com/probot/probot) that assigns all PRs review to a specified person (admin)

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
